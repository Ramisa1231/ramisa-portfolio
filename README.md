# Syeda Ramisa Fariha — Applied AI Portfolio

A responsive, accessible portfolio presenting applied-AI engineering work, research, and stakeholder-facing delivery.

## Highlights

- Flagship CouncilQ case study with an architecture view
- Honest project context: independent, research, competition prototype, or workshop extension
- Responsive layout and keyboard focus states
- Open Graph metadata and semantic HTML
- No build step or framework dependency

## Run locally

Open `index.html` directly, or serve the folder locally:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a public repository named `ramisa-portfolio`.
2. Place the files from this directory at the repository root.
3. In **Settings → Pages**, select **Deploy from a branch**.
4. Choose `main` and `/ (root)`.

The default URL will be:

`https://ramisa1231.github.io/ramisa-portfolio/`

## Before publishing

- Add a real Open Graph preview image and `og:image` tag.
- Add a résumé PDF only if the public version is current and privacy-safe.
- Add live-demo links when deployments are stable.
- Replace qualitative claims with measured results as project benchmarks mature.
